# This project is discontinued as a quest book has been added ingame.

A web-based quest tracking tool for the game **Dark and Darker**. Track quest progress, organize by dungeon, and manage quest completion status.

## Features

- **Quest Tracking**: Track individual quest tasks with checkboxes
- **Dungeon Organization**: Quests are organized by dungeon (Any, Goblin Caves, Ice Caverns, Howling Crypts, Inferno, Abyss)
- **Progress Persistence**: Your progress is automatically saved to browser localStorage
- **Completed Quests**: Automatically moves completed quests to a collapsible section
- **Reset Functionality**: Reset all quest progress with a single button
- **Visual Indicators**: Icons and map images to help identify quest types and locations

## Usage

1. Visit the [Online Browser Version](https://goodgui.github.io/Dark-and-Darker-QuestLog/) or open `index.html` in your web browser
2. Check off tasks as you complete them in-game
3. Completed quests will automatically move to the "Completed Quests" section
